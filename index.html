padding and Margin

the margin and padding classes are maked to have some kind of concistence on that property every time we use it

how to use it:

all the clses use the fallowing structure (property)-(side)-(value) where:

Property can be:

  • margin
  • padding

side can be:

  • left
  • right
  • top
  • bottom
  • x for the horizontal axis
  • y for the vertical axis
