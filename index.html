padding and Margin

the margin and padding classes are maked to have some kind of concistence on that property every time we use it

how to use it:

all the clses use the fallowing structure (property)-(side)-(value) where:

Property can be:

  • margin
  • padding

Side can be:

  • left
  • right
  • top
  • bottom
  • x for the horizontal axis
  • y for the vertical axis

Value can be:

  • 0 for 0 value
  • 1 for .25rem value
  • 2 for .5rem value
  • 3 for 1rem value
  • 4 for 1.5rem value
  • 5 for 2.3 rem value

and a special class is the margin-x-auto who set the left and the right margin to auto, for those elements who has display-block and have a specific width setted

example

        .padding-2 {
          padding: .5rem;
        }
        .margin-top-2 {
          margin: .5rem;
        }
        .margin-y-2 {
          margin-top: .5rem;
          margin-bottom: .5rem;
        }
        .margin-x-auto {
          margin-left: auto;
          margin-right: auto;
        }
